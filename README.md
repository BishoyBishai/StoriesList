# Stories List

Stack

- React 16.9
- Typescript 3.6
- Axios (for ajax call)
- Jest &  Enzyme

# Features

- Loading stories ids from external API 
- Loading stories data as small patches 
- Infinite Scrolling
- Test cases 


### Installation

Install the dependencies and devDependencies
For Frontend

```sh
$ npm install
$ npm run dev
```

For testing

```sh
$ npm run test
```
